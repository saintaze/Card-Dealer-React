# Card-Dealer-React
 A simple app to deal playing cards.

### Features
+ Beautiful UI
+ Card placement animation

##### Built With
+ HTML5
+ CSS3
+ React

### Getting Started
clone repo

`cd` into folder and run `npm install`

run `npm start`
